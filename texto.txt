Este é um texto de exemplo para a disciplina de arquitetura de computadores na Universidade Federal do Ceará. É a turma do semestre 2021.1. Queremos carregar e exibir na tela a partir de um carregamento em disco. Quardaremos estes dados no setor 2. Para ser reconhecido como uma imagem de disco, o arquivo precisa ter um tamanho múltiplo de 512 bytes, dado que 512 bytes é o tamanho de um setor. Completaremos com espaço os 512 caracteres necessários. Estamos em 467. Estou completando o texto para ficar maior do que 512 bytes.
